  • 当到了链表的第一个位置的时候会先出队，并且指针指向第一个的下一个结点。比如0号A，当A出队的时候指针会指向A的下一个元素，由此p会沿着该链表一直走下去看看是否有未被访问过的元素，如果有则先入队。

235. 二叉搜索树的最近公共祖先

发表于 2022-01-25 更新于 2024-01-24 分类于 Leetcode树专题

题目描述

给定一个==二叉搜索树==, 找到该树中两个指定节点的最近公共祖先。

百度百科中最近公共祖先的定义为：“对于有根树 T 的两个结点 p、q，最近公共祖先表示为一个结点 x，满足 x 是 p、q 的祖先且 x 的深度尽可能大（一个节点也可以是它自己的祖先）。”

例如，给定如下二叉搜索树: root = [6,2,8,0,4,7,9,null,null,3,5]

示例 1:

输入: root = [6,2,8,0,4,7,9,null,null,3,5], p = 2, q = 8
输出: 6
解释: 节点 2 和节点 8 的最近公共祖先是 6。
示例 2:

输入: root = [6,2,8,0,4,7,9,null,null,3,5], p = 2, q = 4
输出: 2
解释: 节点 2 和节点 4 的最近公共祖先是 2, 因为根据定义最近公共祖先节点可以为节点本身。

说明:

所有节点的值都是唯一的。
p、q 为不同节点且均存在于给定的二叉搜索树中。

思考

😅一开始没有看到是二叉搜索树！走了很多弯路。。。。。这道题应该用到搜索树的性质来做。

我们画张图容易理解！

  • 如果判断两值都小于当前节点往左走
  • 再如果判断两值都大于当前节点往右走
  • 否则当前节点就是要找的节点。再break出去。
