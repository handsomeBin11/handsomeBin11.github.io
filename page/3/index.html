遍历算法

大体思路：当你建立好邻接链表的时候可以观察到，其实就是一个大数组把一个个小数组串起来。如下图：

  • 当到了链表的第一个位置的时候会先出队，并且指针指向第一个的下一个结点。比如0号A，当A出队的时候指针会指向A的下一个元素，由此p会沿着该链表一直走下去看看是否有未被访问过的元素，如果有则先入队。

  • 如果说该链表走完了，则从队列中出队列一个元素，例如，处理完A后，B会出列，那么指针会走到B的下一个位置遍历整个链表。接下来会如此往复，直到全部链表都被访问过。

给你一个二叉搜索树的根节点 root ，返回 树中任意两不同节点值之间的最小差值 。

差值是一个正数，其数值等于两值之差的绝对值。

示例 1：

输入：root = [4,2,6,1,3]
输出：1
示例 2：

输入：root = [1,0,48,null,null,12,49]
输出：1

提示：

树中节点的数目范围是 [2, 104]
0 <= Node.val <= 105

解答

根据提示，该树是一个二叉搜索树。所以很容易想到利用性质来。

二叉搜索树的中序遍历结果为一个有序数列。

所以思路是:

  • 创建一个vector 来存储遍历后的结果集。
  • 常规的二叉树的中序遍历函数结果保存到事先创好的数组中。
  • 遍历数组 利用temp零时保存后一个减前一个的结果。
  • 不断刷新min值直到找出min。
