有关Linux系统的命令

发表于 2020-04-27 更新于 2024-01-24 分类于 Linux

挂载命令

光盘有设备文件名和挂载点链接起来的过程

查询于自动挂载

  • mount 查询已经挂载的设备
  • mount -a

挂载光盘

  1. 先创建一个挂载点 mkdir /mnt/cdrom
  2. mount /dev/cdrom /mnt/cdrom 把/dev/cdrom 挂载到 /mnt/cdrom（挂载点） 访问的时候当然访问挂载点 也就是我门所说的盘符

弹出光盘

  • umount /dev/sr0

查看U盘设备文件名

  • fdisk -l
  • Mount -t vfat /dev/sdb1 /mnt/usb/
  • 弹出 umount .dev.sdb1

进程管理

jobs 显示工作号 -l 显示进程号

  • fg % 【工作号】 ——–将后台暂停的工作恢复到前台执行

  • bg % 【工作号】 ——–将前台暂停的工作恢复到后台执行

  • ctrl + c 中断程序执行

  • ctrl+z 切换

ps 命令显示进程状态

  • -a 显示终端所有
  • -l 显示进程详细
  • -w 显示完整信息 通过换行把完整信息显示出来
  • -r 正在显示运行当中的进程
  • -u 显示用户的进程ps
  • -au 经常用
  • -aux | grep vi

kill 终止正在运行的进程或作业 + 进程号

  • -l 显示系统定义的进程信号

  • kill -9 + (进程号PID)

  • all + 程序名 一次性的杀死所有指定程序

nice 用于设置优先级

  • -n 19 vi temp

who 查看当前登陆的用户

-H 显示标题

top 查看进程动态监控状态 而 ps 是静态的 是一个时间点上的

  • 按q退出
  • -d 设置间隔时间
  • -p 只显示制定的

vi +名称 创建vi编辑器

进程调度管理命令

  • at

  • atq 查询未执行的作业

  • at -d + 作业号删除已经安排好的作业

  • dmesg 开机时内核检测信息查

系统资源查看

  • uname 内核信息
  • free 查看内存和交换分区
