自增自减运算符的几种用法

Posted on 2020-11-15 Edited on 2024-01-24 In C语言理论部分

初阶 ++i 与 i++ 问题

先设两个值相同的变量

分解：

  1. p = i
  2. i = i+1

分解:

  1. j = j+1;
  2. y = j;

我们来看看结果!

结果是截然不同的，原因是：在 int p = i++; 的时候 第一步就先赋值给了p变量，然后i才叠加的1。所以p变量是没有任何变化的，有变化的应该是那个i，我们打印出来看看。

进阶 *p1++ 和 *++p2

定义指针类型的变量

分解：

  1. p = *p1
  2. p1 = p1 + 1

分解：

  1. p2 = p2 + 1 地址指向下一个单元
  2. q = *p2

看看结果！

为何不同？ int p = p1++ 首先将**p = p1 赋值给了p，再将p1地址向后移一位。故对p1当前输出的那个p1没有任何作用，输出之后的p1才起了作用，我们打印出来看看
