Python 列表元祖字典

列表

通俗来说，列表==其他语言中的数组

特性： 存储相同元素类型的数据 (虽然能存不同的)

列表的定义

添加元素的方法

元组

元祖的特性就是一旦定义了就不可更改性，主要运用于不同数据类型的存储。

定义形式也与列表不同, 是用（）来定义的

应用场景一般有三个

  1. 防止别人乱改list中的元素所以转换成turple
  2. print中的格式化输出实际上是个元祖
  3. 函数返回中用元祖可以反回多个类型的值

字典

实际上就是一个具有key 和 vlaue 的一个描述一个具体事务的一个东西（哈希表）
