插入排序详解

Posted on 2021-02-19 Edited on 2024-01-24 In 排序算法

超详尽解释：

  • if ( L->r[i] < L->r[i-1] ) 因为默认第一个数字已经排好了，所以拿后面的跟前面的比较，如果说后面比前面的还要小，那么就继续操作
  • L->r[0] = L->r[i]; 先把后面那个（也就是小的那个）给了哨兵，因为我们一会前面的元素要替代掉它。
  • for (j = i-1; L->r[0]r[j] ; j–) 其中 j = i-1 是 我要从前面的那个元素开始往后要替换 L->r[0]r[j]相当于说当前这个元素要以哨兵为底线往前走，如果说前面的都大那么该元素就应该放在第一个位置恰好哨兵能作为结束的标志（也就是说，该元素比任何都小的情况下不可能比自个还小所以应该是自己等于自己 所以L->r[0]r[j]） 判断条件就不满足了！哨兵其实也就是你当前拿出来的哪张牌，如果比前面的直接大了那么直接条件不成立循环进不去 j– 的目的也就是说，要不断的往前进行比较。也就相当于 j 和 j-1 之间的比较。
  • L->r[j+1] = L->r[0]; 当循环出来之后，代表前面那个已经比后面这个小了。又因为上一个循环导致 j– 了，而 j– 那个位置上恰好是不满足的（也就说 比该元素小的） 那就应该把该元素放到 j+1 （往后放一格子的位置)也就是在他后面。
