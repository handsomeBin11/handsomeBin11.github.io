为蛇么有结构体？

如果你想连续存放一些数据，你可以用数组，但前提条件是数组必须每个元素的值都是相同类型的

但如果你想存放一些不同类型的数据，把它们弄成一个整体—-出现了结构体。

声明在函数外还是内？

  • 如果你想只在某个函数里使用，就定义在函数里，就像上面的main函数里一样。

  • 如果你想在程序的每一处都能够使用，那么你就定义在外面如下：

声明结构的形式–3种形式

  • ~~~c
    struct point{
    int x;
    int y;
    };

    struct point p1,p2;

  • ~~~c
    struct point{
    int x;
    int y;
    }p1,p2;
    //相当于说，定义的时候顺便定义了 p1 p2
    //以后你想用该结构变量的时候，还是可以采用 struct point *** 的形式来定义的

如图，当你花括号里这样定义的时候，编译器会自动帮你一一对应赋值结构中的变量。

如果你想指定的几个赋值的话！采用 .神马 的方式来进行赋值

当然，你没有赋值的变量会像数组那样，自动赋 0 值

访问结构中的成员

结构和数组的情况有点像

  • 数组用下标的方式访问里面的成员

    • arr[ 3 ] = 10;

  • 结构 用 . 运算符 和 名字来访问其成员

    • today.day
    • student.Name
    • p1.x
    • p1.y

结构运算

  • 要访问整个结构，直接用结构变量的名字

  • 结构可以做

    • 赋值

    • 取地址

    • 传递给函数的参数

    • ~~~c
      p1 = (struct point){5,10};//强制转换
      //相当于 p1.x = 5 p1.y = 10

    结构指针

    注意⚠️，结构的名称不是变量的地址（这里和数组是不同的），所以说加要取地址运算符。

结构与函数

  • 整个结构是可以作为参数传入函数的
  • 这时候，是在函数内新建了一个新的结构变量（克隆体），然后把调用者的值复制进去
  • 也可以返回一个结构
  • 这与数组完全不同

嵌套的结构

什么意思呢？

  • 首先我们定义了一个struct point 类型。该类型里有 x 和 y 两个变量。
  • 实际含义是 整个point 能表达一个点，就是坐标轴 x 和 y 。
  • 我们又定义了一个struct rectangle（矩形） 类型 。该类型里有两个点。pt1 和 pt2
  • 因为两个点就能确定一个矩形！！所以我们定义这个变量只要考虑两个成员即可。
  • 定义struct rectangle 类型的变量 r。
  • r.pt1.x 就代表 rectangle 里面的一个点r.pt1的x轴坐标为…..
  • r.pt2.y 就代表 rectangle 里面的一个点r.pt2的y轴坐标为…..

结构变量定义的几种等价形式

变量定义

如下的4种形式都是等价的

  • r . pt1 . x —–常规写法

  • rp -> pt1.x

  • 经过 rp = &r 之后，rp得到了r的地址

  • 此时 rp 就是那个 r

  • 所以有 rp 所指的那个pt1的 x（rp->pt1.x）

  • rp -> pt1.x ==== r . pt1 . x

  • (r . rpt1 ) . x

  • (rp -> pt1).x
