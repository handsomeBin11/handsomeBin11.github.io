如何得到那个地址？

我们画个图来看看究竟发生了什么？👇

  1. i 是int类型的
  2. p 存放的是 i 的地址，p就应该是 int *类型的。即p能存放 int 类型变量的地址。
  3. q 存放了 p的地址， p又是 int * 类型的。故，q 能存放是 int 类型指针变量的地址。所以是 (int * ) *
  4. r 存放了 q的地址， q又是 能存放 int ** 类型的，故，r能存放 int 类型变量(p)的指针变量（q）的指针变量的地址。
