求给定精度的简单

发表于2021-04-27|更新于2021-04-29|C语言编程题
|阅读量:

本题要求编写程序，计算序列部分和 1 - 1/4 + 1/7 - 1/10 + … 直到最后一项的绝对值不大于给定精度eps。

  • 很明确，题目说，直到最后一项的绝对值不大于给定精度eps。
  • 那我就得定义每一项 也就是那个item
  • 观察其中的每一项，分子都是1 ，分母公差为3
    1. item = flag * 1.0 / den ——> 其中 flag是正负号， 注意分式要用下1.0浮点数。
    2. 把当前的这一项加到 sum里面去。
    3. flag = -flag —–> 正负号交替。
    4. 分母要每一轮 + 3

遇到分式，分子为1 的情况。考虑先把一个分式求出来，只考虑分母即可。分母的初值由题中给出。

只需要在题给出的值赋初始值即可。
