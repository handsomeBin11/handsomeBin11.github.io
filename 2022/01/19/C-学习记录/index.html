##5.1 指针

5.1指针所占内存空间

由此结果可见，不论是什么类型的指针变量。所占的内存空间均为8（我的 电脑是64位）

5.2 空指针/野指针

5.2.1空指针
  1. 空指针用来给指针初始化——–> 也就是你不知道指针该指向哪的时候先指向空
  2. 空指针不能被访问！如果像上图那样，试图解引用来访问会崩溃。
5.2.2野指针

相当于不是你开的房间你进入非法的领域！！瞎指了一块地址就要访问。

5.3 const修饰的指针

5.3.1 const 修饰指针 –常量指针

  • 不能修改指针所指向的值
  • 但是可以修改指针的指向

5.3.2 const 修饰常量 – 指针常量

  • 可以更改指针所指向的值✅
  • 不可以更改指针的指向❌

const 后面跟着int * 那么取 的操作就不允许做了*

const 后面跟着指针变量 那么指针的所指向就不能改了。

5.3.3 const 既修饰指针也修饰变量

  • 所指的值 和 指向不能改

主函数中 姓名：张三 年龄： 22 分数：99

充分证明了，虽然在子函数function1 中我们修改了张三的年龄，==但是在主函数观察并其实没有修改。==

在子函数中只是主函数stu变量的一种copy形式，==并不是真正的它。==

  • 情况二

    子函数中 姓名：张三 年龄： 250 分数：99
    主函数中 姓名：张三 年龄： 250 分数：99

    发生了真正意义上的修改！这个就是stu的真身了，子函数中修改过后，在主函数中的输出了同样的内容。

    ==说明我们发送了地址之后，真正意义上的修改了stu。==
