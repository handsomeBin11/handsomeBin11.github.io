数据库查询总结

数据库对于我来说很难受，或许是我天生对表格不敏感吧。。。导致我学起来很是费劲，不过whatever!让我们来总结几个常用语句吧！

在使用某库的时候别忘了要use 库名哦～

基础查询

语法：select 查询列表 from 表名

特点是：

  1. 查询列表可以是：表中的字段，常量值，表达式，函数
  2. 查询的结果是一个虚拟的表格
查询表中的某个字段

SELECT 字段1，字段2，字段n from 表名

查询所以字段

SELECT * from 表名

起别名

Select last_name 姓,first_name 名 from 表

Select last_name as 姓,first_name as 名 from 表

去重 distinct

例如：查询员工表中涉及到的所有的部门编号

很多员工来自相同部门所以去除重复, distinct要放在select后字段前

Select distinct department_id FROM employees;

编码，隐匿在计算机背后的语言详解

Posted on 2023-07-07 Edited on 2024-01-24 In 书籍

1.一切都应该从电说起

关于电仍然存在着许多未解之谜。如果我们把一个💡灯泡和电池🔋还有一根导线连成一个闭合回路，我们称为电路 。

如果我们把世界上所有的物体无限缩小，可以看到其实是由原子（atom）构成的ps: 当然还可以继续分下去这里我就不继续追溯下去了 原子的结构包括为：质子，中子，电子。如下图所示，

质子带正电 中子不带电都是处在中心的区域（像是银河系中的太阳一样） ，而带电的是核外的电子。像是绕太阳运行的行星。

尽管质子和电子通常用+，- 号表示，但不表示任何的数学含义，只表示某种特殊的关系。通常状况下它们表现为稳定状态。当你从地毯走过的时候，地毯偷偷从你的写上偷走电子，在你接触物体的那一瞬间产生静电火花🔥！在产生一瞬间之后，趋于平衡状态……。在这个过程中是一个回路，电子从你的地毯—>身体—> 鞋子。这种现象不仅仅存在于静电中，在风暴中，底层云积聚了大量的电子，而顶层的失去了大量的电子，然而一到闪电过后，一切趋于平衡….

在电路中，当某个原子中的电子逃逸到下一个原子中，于此同时，该原子又从上一个原子中得到一个电子，而失去电子的原子又从上一个原子获取电子，如此循环，电流其实是电子不断从一个原子移动到另一个原子的过程。

这一切肯定不是自发的，当你把一堆材料堆在一起一定不会产生电流。显然需要一个”驱动器”，那就是电池，电池会促进电路中电子的流动，从而产生电流。

如图，环形电路中，红色表示电流。两个电池，没个电池为1.5v。注意，电池的摆放必须是正负极对接。原因是负极的电子要流向正极产生电流。然而，两个1.5v 串联结果是3v。

判断物体是否容易放电其实是看核外电子是否容易被失去，也就看是不是稳定。比如我们经常用的铜作为导线，是因为铜核外只有一个电子，容易失去电子，也即是容易逃逸！判断是否为导体就是看是否容易逃逸电子。 相反，导电相反是阻抗性，就是一些物质容易让电子通过，称为阻抗性。绝缘体就是这么来的。但是，大力出奇迹 : 其实只要电流足够大任何物体都可以导电 。

铜的阻抗性就比较低，但铜线越长阻抗性会越高。如果铜线越粗，阻抗性会越低，也就是可以流过的电子数量会越高。（类比水管） 电流就相当于水管中水的流速，水压相当于电压，水管的粗细程度代表电阻。

电压： 比如一块1.5v的电池，说明电压是1.5v。电压表示了电流做功的势，也就是电势能的大小。就算不用这块电池，它的电势能依旧在那。势能好比是你拿一块砖在平地上，此时的势能几乎为0。但如果你把它拿到18层楼高的地方（但你别真的做！），此时的势能绝对挺大，就算你什么也不做它的势能依旧在那里。如果你想知道势能到底是多大，你得把它从18层释放下来…..

​ 欧姆定律： I = E/R

在电学中，如果知道电势能和电阻，就可以算出电流的大小。

既然在天空中会打雷，那么空气也是一种导体。把电池放到空气中应该也会有从正负极有电流产生。但是！！根据公式可以看出：

  • 当放在空气中虽然知道电势是1.5v ，但空气中电阻是非常非常大的。故是 一个数/∞ = 0 ，最后算出电流是趋近于0。
  • 当单独连个导线，想必初中的时候讲过，会出现短路现象。原因是当电势能1.5的时候，正负极连着一根导线，单独的一根导线导致电阻值非常的小。所以是，一个数/趋紧于0的数 = ∞ 电流无穷大！！ 意味着导线可能电流过大，导致整个导线变热甚至会烧红（甚至融化），会把电能转换为热能。

大部分电路其实是介于这二者之间的，要么阻值大一点，要么小一点。当阻值小的时候，电路会发热，甚至燃烧。💡灯其实是利用了这个原理，只不过是在真空中然后，灯丝其实会发热然后燃烧。

以上其实不是最主要的，开关才是最想说的。开关表示了电路的 通/闭。其实对应了二进制的 0 /1 。

2.绕过拐角的通信

代入一个情景，你跟你的邻居好友在被妈妈没收手机后想在夜晚睡不着的时候进行交流。 但是，可是你们的窗户是错位的，也就是不能看到对方的窗户。所以不能用手电开关及其用摩斯码通过 点和划 来解读对方的话语。

怎么办？

已经通过之前的电路知识，给对方搭建一套“电报”系统。

但这个只是单向传递，你只能让对方看见，如果你想双向传递，你得再加一套通往你家的。

🤔如果再想想，其实可以省点线路“钱”，比如像这样，中间蓝色部分用公共线。如果你朋友那边闭合开关，通过图中的箭头的方向可以形成一个闭合回路会点亮你家里的灯泡💡。

如果你们俩都闭合开关那么灯泡其实是不亮的。原因是在公共线路中有两个方向相反的电流。两端都没有办法形成闭合回路。

如果我们铺设的线路足够长，对应的电阻也会无限大。这显然不符合经济性。

其实还可以继续简化公用线！减少用线成本。我们知道一个理论，导体的截面越大，导电性就越好。我们这恰好有一个大球，直径大约7900英里，那就是地球呀🌍。地球是个很好的导体，如果你使用地球作为“导线”，那你的电压可不能只是两节干电池了，你需要大点的电压。然后插根铜柱子根地面有足够的接触面积。

由于不能使用干电池了，现在把电池那块抽象成一个符号方便后续的描述。

把V可以想象成一台能抽电子的吸尘器，整个地球🌍其实是一个电子的海洋，有无穷无尽的电子可以使用。

那可以把上面的电报系统简化成现在这种样子。

##【问题描述】

小明决定从下周一开始努力刷题准备蓝桥杯竞赛。他计划周一至周五每天

做 a 道题目，周六和周日每天做 b 道题目。请你帮小明计算，按照计划他将在

第几天实现做题数大于等于 n 题？

【输入格式】

输入一行包含三个整数 a, b 和 n.

【输出格式】

输出一个整数代表天数。

【样例输入】

10 20 99

【样例输出】

8

思路

  • 算出一周 到 周五 5 天做a道，周末做 b 道。 那么我们可以先算出每周能做多少道。

  • 知道总共能做多少道，然后拿总的目标 n / 每周能做多少道 可以算出最多能做几周 。然后 乘 7 代表最多能做多少个整周的天数。

  • 然后 n % 每周能做多少道 == 余下几个零散的天数，然后下一步进行分类。

  • 用一个循环在 多少个整周的天数的基础上一天一天加

  • 如果 等于 6 ，7 的时候就在余数里面扣出 b 天

  • 如果 不是的话 就扣出 a天。

2022蓝桥杯真题《顺子日期》

Posted on 2022-04-10 Edited on 2024-01-24 In 蓝桥杯,普通类型

题目描述

小明特别喜欢顺子。顺子指的就是连续的三个数字：123、456 等。顺子日

期指的就是在日期的 yyyymmdd 表示法中，存在任意连续的三位数是一个顺

子的日期。例如 20220123 就是一个顺子日期，因为它出现了一个顺子：123；

而 20221023 则不是一个顺子日期，它一个顺子也没有。小明想知道在整个 2022

年份中，一共有多少个顺子日期。

解答

思路： 先搞定后四位弄成字符串组合上2022 然后对于整体进行检查是否满足条件

  • 先解决后四位的问题，20220123。月好说，每个月有多少天事先定义一个const的数组出来，一会 循环调用即可，省去了在代码中构思一个月有几天的问题。
  • 后四位如何解决？ 弄一个函数，将月和日拆分送进去。构成字符串，然后检查，如果两位，那就反转一下成为正常顺序，如果是一位前面补0。
